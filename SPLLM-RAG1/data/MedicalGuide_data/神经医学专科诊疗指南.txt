神经医学专科诊疗指南

一、头晕的诊断与鉴别

1. 头晕的分类
- 前庭性眩晕：天旋地转感，常伴恶心、呕吐
- 非前庭性头晕：头昏、头重脚轻、不稳感
- 晕厥前状态：黑朦、视物模糊、濒死感
- 平衡障碍：步态不稳、易跌倒

2. 常见病因
2.1 中枢性病因
- 脑梗死/脑出血
- 短暂性脑缺血发作（TIA）
- 后循环缺血
- 脑干病变
- 小脑病变
- 多发性硬化

2.2 外周性病因
- 良性阵发性位置性眩晕（BPPV）
- 前庭神经炎
- 梅尼埃病
- 听神经瘤

2.3 全身性病因
- 心源性：心律失常、心功能不全
- 血压异常：低血压、直立性低血压、高血压急症
- 贫血
- 低血糖
- 甲状腺功能异常
- 药物副作用

3. 问诊要点
3.1 症状特征
- 发作性还是持续性？
- 持续时间：秒、分钟、小时、天？
- 天旋地转（真性眩晕）还是头昏沉沉（假性眩晕）？
- 体位变化是否诱发或加重？

3.2 伴随症状
- 耳鸣、听力下降（提示外周性）
- 恶心、呕吐
- 肢体无力、麻木（提示中枢性）
- 言语不清、吞咽困难
- 头痛
- 视物成双、视力下降
- 心悸、胸闷

3.3 危险因素
- 高血压、糖尿病、高血脂
- 心脏病
- 既往脑血管病史
- 用药史（降压药、镇静药）

4. 体格检查要点
4.1 生命体征
- 血压（包括直立性血压试验）
- 心率
- 体温

4.2 神经系统检查
- 颅神经：特别注意眼球运动、瞳孔、面部感觉和运动
- 肢体肌力、肌张力
- 共济运动：指鼻试验、跟膝胫试验
- 步态
- 病理征
- 眼震检查

4.3 专科体检
- Dix-Hallpike试验（BPPV）
- 变换体位试验
- Romberg征

5. 辅助检查
5.1 必要检查
- 血常规、血糖
- 肝肾功能、电解质
- 心电图

5.2 病因筛查
- 颈动脉超声
- 经颅多普勒（TCD）
- 头颅CT或MRI
- 前庭功能检查

5.3 特殊检查
- 24小时动态血压
- 动态心电图
- 听力测试

6. Red Flags（危险信号）
- 急性起病伴有神经功能缺损（肢体无力、言语不清、吞咽困难）
- 剧烈头痛
- 意识障碍
- 发热伴头晕
- 新发复视
- 共济失调明显
- 听力突然丧失

7. 常见诊断
7.1 脑梗死后头晕
- 特点：持续性头晕、乏力，可能伴步态不稳
- 多有高血压、糖尿病等危险因素
- 头颅影像学可见梗死灶
- 治疗：抗血小板、控制危险因素、康复训练

7.2 后循环缺血
- 特点：反复发作性眩晕、头晕
- 可能伴视物模糊、复视、肢体无力
- TCD或MRA可见椎基底动脉狭窄

7.3 良性阵发性位置性眩晕
- 特点：体位改变诱发的短暂性眩晕（<1分钟）
- Dix-Hallpike试验阳性
- 治疗：手法复位

7.4 前庭神经炎
- 特点：急性起病，持续性眩晕数天
- 无听力下降
- 治疗：对症+前庭康复

二、乏力的诊断与鉴别

1. 乏力的分类
- 全身性乏力
- 局部肌肉无力
- 易疲劳

2. 常见病因
2.1 神经肌肉疾病
- 脑血管病
- 重症肌无力
- 周围神经病
- 肌炎

2.2 全身性疾病
- 贫血
- 甲状腺功能减退或亢进
- 肾上腺皮质功能不全
- 糖尿病
- 肝肾功能不全
- 心功能不全
- 慢性感染
- 肿瘤

2.3 心理因素
- 抑郁症
- 焦虑症
- 慢性疲劳综合征

3. 问诊要点
- 发生时间、持续时间、进展速度
- 活动后是否加重、休息后是否缓解
- 是否伴有其他症状：头晕、心悸、消瘦、发热
- 用药史、生活习惯
- 睡眠情况、精神状态

4. 辅助检查
- 血常规（排除贫血）
- 生化全套（肝肾功能、电解质、血糖）
- 甲状腺功能
- 糖化血红蛋白
- 维生素B12、叶酸
- 肌酸激酶
- 必要时：头颅影像、肌电图

三、脑梗死的诊疗

1. 诊断标准
1.1 临床表现
- 急性起病
- 局灶性神经功能缺损症状
- 症状持续24小时以上（或导致死亡）

1.2 影像学
- 头颅CT：早期可能无异常，24-48小时后显示低密度灶
- 头颅MRI：DWI序列早期即可发现高信号

1.3 分型
- 大动脉粥样硬化型
- 心源性栓塞型
- 小动脉闭塞型（腔隙性梗死）
- 其他明确病因型
- 不明原因型

2. 危险因素
2.1 不可控因素
- 年龄、性别、种族
- 遗传因素

2.2 可控因素
- 高血压（最重要）
- 糖尿病
- 高脂血症
- 心房颤动
- 吸烟
- 酗酒
- 肥胖
- 缺乏运动

3. 急性期治疗
3.1 溶栓治疗
- 时间窗：发病4.5小时内
- 药物：rt-PA
- 需排除禁忌症

3.2 抗血小板治疗
- 首选：阿司匹林 100-300mg/d
- 或氯吡格雷 75mg/d
- 急性期可短期双抗

3.3 降压治疗
- 急性期血压管理：允许血压升高
- 不合并溶栓：<220/120 mmHg时不降压
- 合并溶栓：维持<180/105 mmHg

3.4 其他治疗
- 改善脑血流灌注
- 神经保护
- 控制血糖
- 降温（高热时）
- 防治并发症

4. 二级预防
4.1 抗血小板治疗
- 长期服用阿司匹林或氯吡格雷
- 非心源性栓塞患者

4.2 抗凝治疗
- 心源性栓塞患者
- 华法林或新型口服抗凝药（NOAC）

4.3 控制危险因素
- 降压：目标<140/90 mmHg（糖尿病或肾病<130/80）
- 降糖：HbA1c<7%
- 调脂：LDL-C<2.6 mmol/L（高危<1.8）
- 戒烟限酒
- 控制体重
- 适度运动

4.4 他汀类药物
- 所有缺血性卒中患者均应服用
- 阿托伐他汀 20-40mg/d
- 或瑞舒伐他汀 10-20mg/d

5. 康复治疗
5.1 早期康复
- 病情稳定后48小时即可开始
- 肢体被动活动、按摩
- 良肢位摆放

5.2 恢复期康复
- 物理治疗：运动疗法、理疗
- 作业治疗：日常生活能力训练
- 言语治疗：失语症训练
- 吞咽训练
- 心理治疗

6. 随访管理
6.1 随访频率
- 发病后1个月内：每周1次
- 1-3个月：每2周1次
- 3-6个月：每月1次
- 6个月后：每3个月1次

6.2 监测指标
- 症状变化
- 血压、血糖、血脂
- 肝肾功能（用药监测）
- 影像学复查（必要时）

6.3 复诊指征
- 症状复发或加重
- 新发症状
- 药物副作用
- 指标控制不佳

四、高血压的管理

1. 诊断标准
- 非同日3次测量，SBP≥140 mmHg 和/或 DBP≥90 mmHg

2. 分级
- 1级（轻度）：140-159/90-99 mmHg
- 2级（中度）：160-179/100-109 mmHg
- 3级（重度）：≥180/110 mmHg

3. 降压目标
- 一般人群：<140/90 mmHg
- 合并糖尿病或肾病：<130/80 mmHg
- 老年人（≥65岁）：<150/90 mmHg，能耐受可<140/90

4. 药物选择
4.1 一线药物
- ACEI/ARB：首选（尤其合并糖尿病、肾病）
- CCB（钙通道阻滞剂）
- 利尿剂
- β受体阻滞剂（合并冠心病）

4.2 联合用药
- 2级高血压起始即可联合用药
- 常用组合：ACEI/ARB + CCB
- ACEI/ARB + 利尿剂

4.3 注意事项
- 从小剂量开始
- 逐步增加剂量
- 避免骤然停药
- 监测副作用

五、糖尿病的管理

1. 诊断标准
- 空腹血糖≥7.0 mmol/L
- 或OGTT 2小时血糖≥11.1 mmol/L
- 或HbA1c≥6.5%
- 或典型症状+随机血糖≥11.1 mmol/L

2. 控制目标
- HbA1c<7%（一般成人）
- 空腹血糖 4.4-7.0 mmol/L
- 餐后2小时血糖<10.0 mmol/L

3. 治疗原则
3.1 生活方式干预
- 医学营养治疗
- 运动治疗
- 体重管理
- 戒烟限酒

3.2 药物治疗
- 二甲双胍：首选
- 磺脲类、格列奈类
- α-糖苷酶抑制剂
- DPP-4抑制剂
- GLP-1受体激动剂
- SGLT-2抑制剂
- 胰岛素

4. 并发症筛查
- 糖尿病肾病：尿微量白蛋白
- 糖尿病视网膜病变：眼底检查
- 糖尿病神经病变：神经传导速度
- 糖尿病足：足部检查

六、贫血的诊断与治疗

1. 诊断标准
- 男性：Hb<120 g/L
- 女性：Hb<110 g/L
- 孕妇：Hb<100 g/L

2. 分类
2.1 按病因
- 失血性贫血
- 溶血性贫血
- 造血功能障碍

2.2 按细胞形态
- 小细胞低色素性：铁缺乏、地贫
- 正细胞正色素性：急性失血、慢性病
- 大细胞性：巨幼细胞性贫血、肝病

3. 常见原因
- 缺铁性贫血（最常见）
- 慢性病贫血
- 巨幼细胞性贫血（维生素B12或叶酸缺乏）
- 肾性贫血
- 骨髓增生异常
- 溶血性贫血

4. 辅助检查
- 血常规：Hb、RBC、MCV、MCH、MCHC
- 网织红细胞计数
- 血清铁、铁蛋白、总铁结合力
- 维生素B12、叶酸
- 必要时：骨髓穿刺

5. 治疗
5.1 病因治疗
- 去除病因（止血、治疗基础疾病）

5.2 补充治疗
- 缺铁性：口服铁剂（硫酸亚铁）
- 巨幼细胞性：维生素B12、叶酸
- 肾性：促红细胞生成素

6. 贫血与头晕、乏力
- 轻度贫血可无症状
- 中重度贫血出现头晕、乏力、心悸、气短
- 老年人或合并心脑血管病者症状更明显
- 纠正贫血可改善症状

七、肾功能不全的管理

1. 慢性肾脏病分期
- G1: eGFR≥90 mL/min/1.73m²
- G2: eGFR 60-89
- G3a: eGFR 45-59
- G3b: eGFR 30-44
- G4: eGFR 15-29
- G5: eGFR<15（尿毒症期）

2. 常见病因
- 糖尿病肾病（最常见）
- 高血压肾病
- 慢性肾小球肾炎
- 多囊肾
- 药物性肾损害

3. 管理要点
3.1 延缓疾病进展
- 控制血压：目标<130/80 mmHg
- 控制血糖：HbA1c<7%
- 优质低蛋白饮食
- ACEI/ARB类药物保护肾脏

3.2 治疗并发症
- 贫血：促红素
- 高钾血症：饮食控制、药物治疗
- 代谢性酸中毒：碳酸氢钠
- 矿物质骨病：钙、维生素D

3.3 避免肾损害
- 避免肾毒性药物（NSAIDs、造影剂）
- 药物剂量调整
- 充足饮水

4. 转诊指征
- eGFR<30 mL/min/1.73m²
- 进行性肾功能恶化
- 难以控制的并发症
- 准备肾脏替代治疗

八、用药安全

1. 老年患者用药原则
- 从小剂量开始
- 注意药物相互作用
- 定期监测肝肾功能
- 简化用药方案

2. 肾功能不全患者
- 根据eGFR调整药物剂量
- 避免肾毒性药物
- 优先选择不经肾脏排泄的药物

3. 合并用药注意
- 阿司匹林+氯吡格雷：注意出血风险
- 他汀类+纤维酸类：注意肌病
- ACEI/ARB：注意高钾血症、肾功能恶化

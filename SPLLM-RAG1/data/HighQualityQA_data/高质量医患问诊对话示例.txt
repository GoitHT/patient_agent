案例1：头晕问诊（脑梗死后恢复期）

【场景】：52岁女性患者，主诉头晕乏力4天

第1轮问诊：
医生：您好，请问您哪里不舒服？
患者：医生您好，我主要是头晕，浑身没劲儿，大概有四五天了。

第2轮问诊：
医生：头晕和没劲儿是持续一整天，还是时好时坏？
患者：差不多一整天都这样，早上起来就晕乎乎的，没什么力气，到晚上也没见好。

第3轮问诊：
医生：头晕的时候，有没有感觉天旋地转，或者像坐船一样摇晃？
患者：没有那种天旋地转的感觉，也不是像坐船那样晃。就是感觉脑袋一直昏昏沉沉的，有点发蒙，站不稳当，身上也使不上劲儿。

第4轮问诊：
医生：除了头晕乏力，还有其他不舒服吗？比如恶心、呕吐、头痛？
患者：没有恶心呕吐，也不头痛。就是晕乎乎的，没力气。

第5轮问诊：
医生：您之前有什么病吗？比如高血压、糖尿病？
患者：有高血压，不过我已经停药半年了，觉得血压还挺正常的。4天前因为头晕去医院，做了磁共振，说是脑梗，给开了药保守治疗，现在血压是正常了，可这头晕乏力一直不好。

第6轮问诊：
医生：明白了。您4天前确诊了脑梗死，正在恢复期。头晕乏力是脑梗后常见的症状，但也可能有其他原因，比如贫血、血糖问题等。我需要给您做一些检查进一步明确原因。同时，高血压药不能自行停，这是导致脑梗的重要因素。

【问诊技巧总结】：
✓ 开放式问题开始，让患者自由表达
✓ 逐步聚焦症状特征（持续性vs间歇性，真性眩晕vs假性眩晕）
✓ 询问伴随症状
✓ 详细询问既往史和用药史
✓ 最后给予解释和治疗计划

案例2：良性阵发性位置性眩晕（BPPV）问诊

【场景】：58岁女性患者，主诉头晕2天

第1轮问诊：
医生：您好，请问您哪里不舒服？
患者：医生，我这两天老是头晕，晕得厉害。

第2轮问诊：
医生：您说的头晕，是感觉天旋地转，还是头昏昏沉沉的？
患者：就是天旋地转的感觉，特别晕，还想吐。

第3轮问诊：
医生：什么时候开始的？晕的时候持续多长时间？
患者：前天早上开始的。我翻身的时候突然就晕了，特别晕，但也就几十秒，然后就好了。但是只要一动就晕，不动就不晕。

第4轮问诊：
医生：也就是说，您在某个特定姿势时会突然眩晕，比如翻身、低头、抬头？
患者：对对对！就是这样！一翻身就晕，低头拿东西也晕，但躺着不动就不晕了。

第5轮问诊：
医生：晕的时候有没有耳鸣、听力下降？有没有一侧胳膊腿没劲、说话不清楚这些情况？
患者：没有这些，就是单纯的晕，耳朵听力都好着

呢。

第6轮问诊：
医生：好的，您这个情况很可能是'耳石症'，医学上叫良性阵发性位置性眩晕，是内耳的小石头掉到不该去的地方了。我现在给您做一个检查，让您躺下并转头，如果能诱发出眩晕，就基本确诊了。确诊后我可以给您做一个'耳石复位'的治疗，效果很好，不用吃药。

【问诊技巧总结】：
✓ 明确眩晕类型（真性vs假性）
✓ 抓住关键特征：体位改变诱发、持续时间短暂
✓ 询问伴随症状，排除中枢性病变和其他疾病
✓ 给患者通俗易懂的解释
✓ 告知治疗方案，建立信心

案例3：短暂性脑缺血发作（TIA）问诊

【场景】：65岁男性患者，主诉右侧肢体无力30分钟

第1轮问诊：
医生：您好，发生什么事了？
患者：医生，我刚才右边手脚突然没劲儿了，吓死我了。

第2轮问诊：
医生：什么时候开始的？多长时间？现在还有症状吗？
患者：大概1个多小时前吧，当时右手拿杯子突然拿不住掉了，右腿走路也拖着，持续了大概半个小时，现在好了，手脚都有劲了。

第3轮问诊：
医生：当时有没有意识不清？有没有说话不清楚？有没有头晕、头痛？
患者：意识一直清楚的，就是手脚没劲。好像也没说话不清，也不头晕头痛。

第4轮问诊：
医生：您之前有什么病吗？比如高血压、糖尿病、心脏病？
患者：有高血压和糖尿病，一直在吃药。我还抽烟，抽了三十多年了。

第5轮问诊：
医生：以前有没有发生过类似的情况？
患者：没有，第一次。

第6轮问诊：
医生：您这个情况非常需要重视。虽然症状现在缓解了，但这可能是'小中风'，医学上叫短暂性脑缺血发作，是脑梗死的严重预警信号。您现在是高危状态，短期内可能发生真正的脑梗死。我建议您住院观察，完善脑血管检查，及时用药预防。时间就是大脑，早干预早获益。

【问诊技巧总结】：
✓ 快速获取关键信息：症状、时间、持续时间、是否缓解
✓ 排查伴随症状和其他定位体征
✓ 评估危险因素
✓ 强调严重性和紧迫性
✓ 明确告知诊断和治疗计划

案例4：紧张型头痛问诊

【场景】：35岁男性患者，主诉头痛2个月

第1轮问诊：
医生：您好，请问您哪里不舒服？
患者：医生，我最近老是头疼，已经两个月了。

第2轮问诊：
医生：头疼在什么部位？是一侧还是两侧？是跳着疼还是胀痛、紧绷感？
患者：两边的太阳穴和后脑勺疼，感觉像被紧紧箍着，胀胀的。

第3轮问诊：
医生：什么时候疼得厉害？持续多久？休息后会不会好一些？
患者：一般下午疼得比较厉害，有时候能疼一下午，晚上休息后好一点。工作忙、压力大的时候特别疼。

第4轮问诊：
医生：头疼的时候有没有恶心、呕吐？有没有怕光、怕声音？有没有看东西模糊、有闪光？
患者：没有这些，就是单纯的疼，胀疼。

第5轮问诊：
医生：您的工作是做什么的？工作时间长吗？
患者：我是程序员，每天对着电脑十几个小时，经常加班，压力挺大的。最近项目特别多，睡眠也不好。

第6轮问诊：
医生：您这个头痛，从表现来看是'紧张型头痛'，主要是工作压力大、长时间保持一个姿势、精神紧张引起的。这种头痛很常见，不是什么严重的病，但需要调整生活方式。我建议您：工作时每小时起来活动一下，做做颈部放松操，保证充足睡眠，适度运动。疼得厉害时可以吃点布洛芬，但不要经常吃。如果能减轻压力、改善姿势，症状会明显好转的。

【问诊技巧总结】：
✓ 详细询问头痛特征：部位、性质、时间规律
✓ 明确诱发和缓解因素
✓ 询问伴随症状（排除偏头痛、颅内病变）
✓ 了解生活工作情况
✓ 给予通俗解释和具体建议
✓ 强调非药物治疗的重要性

案例5：周围性面瘫问诊

【场景】：42岁女性患者，主诉右侧面部表情异常1天

第1轮问诊：
医生：您好，请问您哪里不舒服？
患者：医生，我今天早上起来发现右边脸不对劲，嘴歪了。

第2轮问诊：
医生：什么时候发现的？之前有什么不舒服吗？
患者：今天早上刷牙的时候发现的，漱口水从右边嘴角流出来，眼睛也闭不严。之前没感觉，就是前几天有点受凉。

第3轮问诊：
医生：右侧额头的皱纹还有吗？能不能皱眉？眼睛能完全闭上吗？
患者：额头皱纹好像浅了，皱眉感觉右边不动。眼睛闭不严，总感觉在漏风。

第4轮问诊：
医生：除了面部，有没有头晕、头痛、胳膊腿没劲、说话不清楚这些情况？耳朵疼不疼？听力有没有下降？
患者：这些都没有，就是脸不对劲。耳朵也不疼，听力正常。

第5轮问诊：
医生：您这个情况是'面瘫'，医学上叫面神经麻痹。从您的症状看，是'周围性面瘫'，可能是受凉或病毒感染引起的，大多数人治疗后可以完全恢复，不用太担心。但需要尽早治疗，主要用激素和营养神经的药，配合康复训练。同时要特别注意保护眼睛，因为眼睛闭不严容易发炎。

【问诊技巧总结】：
✓ 快速识别面瘫症状
✓ 区分周围性vs中枢性面瘫（额纹、眼睑）
✓ 排查其他神经系统症状
✓ 明确诱因（受凉、病毒感染）
✓ 给予通俗解释，强调预后
✓ 强调早期治疗和眼部保护

问诊沟通技巧总结

【开放式问题】：
✓ "请问您哪里不舒服？"
✓ "能详细说说您的症状吗？"
✓ "还有其他不舒服吗？"

【半开放式问题】：
✓ "头晕是天旋地转，还是头昏沉沉？"
✓ "持续一整天，还是时好时坏？"
✓ "什么时候加重，什么时候缓解？"

【封闭式问题】：
✓ "有没有恶心、呕吐？"
✓ "以前得过高血压吗？"
✓ "现在还疼吗？"

【体贴关怀的语言】：
✓ "我理解您的担心..."
✓ "这个症状确实让人不舒服..."
✓ "不用太紧张，我们慢慢来..."

【通俗易懂的解释】：
✓ 避免：脑血管意外、TIA、BPPV等专业术语
✓ 使用：小中风、耳石症等通俗说法
✓ 打比方：像被紧紧箍着、像坐船一样摇晃

【建立信任和依从性】：
✓ 解释病因和机制
✓ 说明治疗方案和预期效果
✓ 强调配合的重要性
✓ 给予鼓励和信心

【危险信号的识别和处理】：
✓ 识别Red Flags
✓ 果断建议住院或转诊
✓ 强调严重性，但避免过度惊吓
✓ 给予明确的行动指导

检查结果解读对话示例

【场景】：患者拿着血常规报告回诊

医生：您的血常规结果出来了。血红蛋白是98，正常值女性应该在110以上，您这个偏低，说明有贫血。

患者：贫血严重吗？

医生：属于轻度贫血，不是特别严重，但可以解释您头晕乏力的症状。贫血的话，血液携带氧气的能力下降，大脑供氧不足，就会头晕、没劲儿。

患者：那怎么办呢？

医生：我给您开两种药：一个是铁剂，补充铁元素；一个是维生素B12，帮助造血。同时您饮食上要多吃瘦肉、动物肝脏、菠菜这些含铁多的食物。服药1-2个月后复查，看贫血改善情况。

患者：好的，那贫血会不会越来越严重？

医生：不会的。您这个贫血程度不重，补充后会慢慢好转。但要找原因，为什么会贫血。女性最常见的是月经量多或者营养不良。如果补充后还不好转，我们再进一步检查。

【解读技巧】：
✓ 先告知结果，明确异常项
✓ 用通俗语言解释临床意义
✓ 关联症状，让患者理解
✓ 给出治疗方案
✓ 预估预后，消除顾虑
✓ 说明复查计划
